# Q-Core Brochure

Static HTML brochure for the AlphaEdge Q-Core course.

## Files

- `index.html`: main brochure entry
- `assets/`: local image assets used by the brochure

## Deploy To GitHub Pages

1. Create a new GitHub repository.
2. Upload everything inside this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your default branch, and choose the `/ (root)` folder.
6. Save and wait for the Pages site to publish.

## Local Preview

Run:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```
